Análise de Churn: Retenção de Clientes em Telecomunicações 
Este projeto consiste em uma análise detalhada dos dados de uma empresa de telecomunicações para identificar os principais motivos que levam os clientes a cancelarem seus serviços (Churn). O objetivo é fornecer insights estratégicos para reduzir a evasão e aumentar o tempo de permanência do cliente.

Sobre os Dados
O conjunto de dados contém informações sobre:

Serviços assinados: Telefonia, múltiplas linhas, internet, segurança online, backup, proteção de dispositivo, suporte técnico e streaming.

Informações de conta: Tempo de contrato (Tenure), método de pagamento, faturamento mensal e total.

Perfil demográfico: Gênero, faixa etária e se possui parceiros ou dependentes.

Tecnologias e Bibliotecas
Python 3.14

Pandas: Manipulação e tratamento de dados.

Matplotlib & Seaborn: Visualização de dados e análise estatística.

Jupyter Notebook: Ambiente de desenvolvimento.

Etapas do Projeto
Limpeza de Dados: Tratamento de valores nulos e correção de tipos de dados (ex: conversão da coluna TotalCharges para numérico).

Análise Exploratória (EDA): Identificação de padrões de comportamento entre clientes que cancelaram vs. clientes ativos.

Visualização de Insights: Criação de gráficos para validar hipóteses de negócio.

Principais Insights
Modelo de Contrato: Clientes com contratos mensais (Month-to-month) têm uma taxa de churn drasticamente superior aos contratos de longo prazo.

Serviços de Valor: A ausência de suporte técnico e segurança online correlaciona-se fortemente com a saída do cliente.

Período Crítico: Os primeiros 6 meses de contrato são os mais vulneráveis, exigindo maior atenção no onboarding.
